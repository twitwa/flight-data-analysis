# ✈️ Flight Data Analysis

A data analysis project exploring 5.8 million real US flights from 2015.

## What I explored
- Which airline has the most delays?
- Which month is the worst for flying?
- Why do flights get cancelled?

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
2015 Flight Delays and Cancellations from Kaggle
